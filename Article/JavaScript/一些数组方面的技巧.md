### reduce (记住设置初始值)

```js
const cartList = [
  {
    name: 'goods1',
    price: 25,
    amount: 3,
    weight: 500
  },
  {
    name: 'goods2',
    price: 27.5,
    amount: 2,
    weight: 300
  },
  {
    name: 'goods5',
    price: 35,
    amount: 3,
    weight: 200
  },
  {
    name: 'goods7',
    price: 127.5,
    amount: 2,
    weight: 100
  }
]
// 计算这个购物车列表的总价，或者商品总重量
// map + reduce

// output -> 409

// 只用reduce完成总价计算
// 总质量雷同
```
